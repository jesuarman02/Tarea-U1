
¿Qué es Display Flex?
Display Flex es una propiedad de CSS que se utiliza para crear diseños flexibles y responsivos en programación web. Permite organizar elementos HTML en un contenedor flexible, ajustando automáticamente el espacio y el tamaño de los elementos secundarios para adaptarse al diseño, ya sea en una fila o en una columna.

**¿Cómo funciona?** 
Cuando aplicas la propiedad a un contenedor, este se convierte en un contenedor flexible que afecta a sus elementos secundarios directos. Los hijos dentro del contenedor flex se distribuyen de manera automática, dependiendo de las propiedades flexibles que establezcas en el contenedor padre y en los elementos hijos.

**Propiedades clave:**

-  Display: flex: Aplica el modelo de diseño flexible al contenedor.
-  Flex-direction: Determina si los elementos secundarios se distribuirán en una fila o en una columna.
-  Justify-content: Alinea los elementos secundarios a lo largo del eje principal, horizontal para filas, vertical para columnas.
-  Align-items: Alinea los elementos secundarios a lo largo del eje transversal.
-  Flex-grow, flex-shrink, flex-basis: Controlan cómo los elementos secundarios crecen, se encogen y su tamaño base.

¿Qué es Display Grid?

Display grid es una propiedad de CSS que se utiliza para crear diseños de cuadrícula bidimensionales en programación web. Permite organizar elementos HTML en una estructura de filas y columnas, ofreciendo un control más granular y preciso sobre la ubicación y el tamaño de los elementos en una página.

**¿Cómo funciona?** 
Al aplicar display grid a un contenedor, este se convierte en un contenedor de cuadrícula que divide el espacio en filas y columnas. Puedes definir el tamaño de las filas y columnas, así como la ubicación de los elementos secundarios dentro de estas celdas.

**Propiedades clave:**

-  Display grid: Aplica el modelo de diseño de cuadrícula al contenedor.
-  Grid-template-columns: Define el tamaño y la cantidad de columnas en la cuadrícula.
-  Grid-template-rows: Define el tamaño y la cantidad de filas en la cuadrícula.
-  Grid-gap: Establece el espacio entre las celdas (filas y columnas) de la cuadrícula.
- Grid-row y Grid-column: Controlan la ubicación de un elemento dentro de la cuadrícula.
